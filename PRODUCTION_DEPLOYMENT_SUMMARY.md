## 🔑 Authentication Explained

Your app uses **JWT (JSON Web Tokens)**:

1. **User registers** with email/password
2. **Backend creates account** in MongoDB with hashed password
3. **Backend generates JWT token** (unique per user)
4. **Frontend stores JWT** in device storage
5. **Every API call** includes JWT in header
6. **Backend validates JWT** - grants/denies access
7. **Chat uses JWT** for real-time authentication

**You don't need API keys** - everything is automatic!
